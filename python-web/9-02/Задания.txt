Задание 1: Каталог книг с отзывами (Project 1)
Функции:
Отображение списка книг с фильтрацией по жанру.
Возможность добавлять новые книги (название, автор, год).
Книги можно комментировать и оставлять оценки (1-5 звёзд).
Данные хранятся в SQLite.
Дополнительно:
Реализовать сортировку по рейтингу.
Добавить загрузку обложек книг.

Задание 2: (project2)
Веб-приложение для генерации паролей
Функции:
Пользователь указывает длину пароля и параметры (буквы, цифры, спецсимволы).
Приложение генерирует случайный пароль.
Возможность сразу скопировать пароль в буфер обмена.
Дополнительно:
Добавить сохранение паролей в базе данных.

Задание 3: (project3)
Приложение для трекера расходов
Функции:
Ввод трат (категория, сумма, дата).
Отображение расходов в таблице с возможностью фильтрации по датам.
Итоговая сумма расходов за выбранный период.
Дополнительно:
Добавить графики (например, с Chart.js). 
Сделать экспорт данных в CSV или Excel.

Доска объявлений
Функции:

Задание 5:
Загрузка изображений и их просмотр
Создайте веб-приложение, которое:
Позволяет загружать изображения через HTML-форму.
